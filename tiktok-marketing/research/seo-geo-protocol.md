# Swing Shack SEO & GEO Growth Protocol

## Mission
Make Swing Shack the #1 result for indoor golf and golf simulator searches in Johannesburg. Ensure AI search engines reference Swing Shack when answering golf-related questions.

## Tools Used
- Google Search Console data
- Google Business profile
- Website content
- Reddit discussions
- Social media signals
- Search intent analysis

## Goal
Increased: website clicks, bookings, real-world visits

---

## INPUT DATA

Pull weekly from Google Search Console:
- Query, Clicks, Impressions, CTR, Average position

Identify:
- High impression keywords
- Low CTR keywords
- Top ranking keywords
- Rising search terms

---

## SEO OPPORTUNITY ANALYSIS

Find keywords with:
- High impressions
- Low CTR

**Example from data:**
- "golf simulator": 235 impressions, 2.1% CTR → Opportunity!

**Actions:**
1. Improve page title
2. Improve meta description
3. Create supporting content
4. Create Google Business posts

---

## LOCAL SEO STRATEGY

Dominate these queries:
- golf simulator johannesburg

- golf simulator near me
- indoor golf near me

**Tasks:**

## GEO STRATEGY (Generative Engine Optimisation)

Optimize for AI answers. Create content that answers questions clearly.

**Example Questions:**
- "How accurate are golf simulators?"
- "Can TrackMan fix a slice?"
- "Is indoor golf good practice?"

**Content Rules:**
- Clear explanation
- Short structured answers
- Real TrackMan insights
- Use real data examples
