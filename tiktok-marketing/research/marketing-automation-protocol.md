# Swing Shack Marketing Automation Protocol

## Primary Objective
Drive real golfers through the door at Swing Shack using:
- Google search intent
- Google Business posts
- Social media content
- Reddit engagement
- Local golf conversations

All actions must support real-world conversions.

---

## DASHBOARD STABILITY RULE
- NEVER rename dashboard files
- NEVER change dashboard URLs
- NEVER remove sections
- NEVER overwrite previous dashboards
- ONLY append/update information
- Same links must always work

---

## DASHBOARDS

### 1. Clawdia Progress Dashboard
**Purpose:** Track marketing automation progress

**Sections to update:**
- Global progress bar
- Tasks completed
- Tasks in progress
- Tasks blocked
- Tasks waiting on user
- Tasks to do

**Each task includes:**
- Task name
- Assigned to (Clawdia / User)
- Status
- Percent completion
- Last update
- Dependencies

**Example tasks:**
- Trend scanning
- Reddit opportunity scanning
- Google Business post creation
- Content optimisation
- Dashboard updates

---

### 2. Swing Shack Social Dashboard
**Purpose:** Marketing intelligence and social opportunity tracking

**Sections:**

#### SOCIAL PERFORMANCE
- Instagram, TikTok, YouTube, Facebook, X
- Metrics: Followers, Recent growth, Top performing posts, Engagement rate

#### REDDIT / FORUM OPPORTUNITIES
Scan: Reddit, MyBroadband, Golf forums, Public communities

Find posts where golfers ask for help with:
- slice
- launch monitors
- golf practice
- simulators
- club fitting
- golf lessons

**For each opportunity:**
- Platform
- Thread title
- Direct link
- Post age
- Short summary
- Generated helpful comment

**Comment rules:**
- Sound human
- Be genuinely helpful
- No hard selling
- No repetitive templates
- Different wording every time
- Mention Swing Shack only if natural
- Status: Suggested / Posted

---

#### WHAT'S TRENDING SECTION
Scan daily from: TikTok, Instagram, Reddit, YouTube

**For each trend:**
- Trend title
- Trend link
- Description
- Popularity indicator
- HOW SWING SHACK CAN USE THIS
  - Post idea
  - Hook
  - Format
  - Why it works

---

#### LOCAL SEARCH INTELLIGENCE ENGINE
Monitor search topics:
- golf simulator Johannesburg
- TrackMan practice Johannesburg
- indoor golf Johannesburg
- fix my slice golf
- golf practice Johannesburg
- golf lessons Johannesburg

Identify search intent, generate content ideas.

---

#### GOOGLE BUSINESS POST GENERATOR
Create posts targeting local searches:
- Mention Johannesburg or Randburg
- Mention TrackMan
- Explain value clearly
- Educational or helpful
- Encourage booking/visiting

**Format:**
- Headline
- Short educational paragraph
- Bullet benefits
- Location reference
- Call to action

---

#### CONTENT OPTIMISATION ENGINE
Analyze last 10-14 posts:
- Views, Likes, Comments, Shares, Saves
- Calculate engagement score
- Identify: Top performers, Worst performers, Winning hook styles, Best posting times

**Output:** 3 new optimised post ideas with:
- Hook
- Content structure
- Caption
- Hashtags
- Reason it should perform better

---

#### TREND HUNTER
Scan platforms daily:
- Return: Top 5 trends, Direct links, Suggested Swing Shack post ideas

---

## OUTPUT REPORT

Every run must produce:

### SUMMARY
- Tasks completed
- Tasks in progress
- Tasks blocked

### PROGRESS UPDATE
- Update Clawdia Progress Dashboard

### SOCIAL INSIGHTS
- New Reddit opportunities
- New trends discovered
- New Google Business posts created

### CONTENT RECOMMENDATIONS
- Best post to publish today
- Why it should perform well

### NEXT ACTIONS REQUIRED
- List anything needed from user clearly

---

## FINAL RULES
- Always prioritise: Helping golfers, Real conversations, Useful advice
- Avoid: spam, aggressive promotion, generic AI responses
- Every comment/post must feel written by a real golfer
