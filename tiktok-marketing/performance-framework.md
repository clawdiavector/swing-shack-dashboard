# Swing Shack TikTok Performance Framework

## Daily Workflow

### 1. INGEST DATA
Pull last 10-14 posts. Capture:
- Views, Likes, Comments, Shares, Saves
- Post time, Hook type, Topic pillar

### 2. CALCULATE PERFORMANCE
- Like Rate = Likes / Views
- Comment Rate = Comments / Views  
- Share Rate = Shares / Views
- Save Rate = Saves / Views
- Engagement Score = (Likes + 2×Comments + 3×Shares + 3×Saves) / Views

### 3. RANK POSTS
- Top 20% = Winning Pattern
- Bottom 20% = Kill Pattern

### 4. DIAGNOSE
- Views high, Engagement low → Hook strong, Content weak
- Engagement high, Views low → Hook weak, Content strong
- Shares high → Turn into series
- Saves high → Convert to checklist style

### 5. IDENTIFY WINNING VARIABLES
- Best hook style
- Best pillar
- Best time posted
- Best format

### 6. CHOOSE ONE VARIABLE TO TEST
Only one: Hook structure | Slide count | CTA | Tone | Depth | Format

### 7. DESIGN NEXT POST
Output: Hook, Angle, Structure, Why it will outperform, What variable is being tested

---

## Rules
- Never repeat identical hook structure twice
- Never ignore top 20% performers
- Always double down on winners with small twists
- Kill patterns that flop twice
- Think like a performance marketer

---

## Final Output Format
1. Top 3 posts and why they won
2. Bottom 2 posts and why they failed
3. Winning pattern summary
4. Today's test variable
5. Next post design
6. Why this will outperform
